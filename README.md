# AutoTrack

Веб-приложение для ведения журнала поездок с авторизацией через Firebase.

## Возможности

- Вход по Email/Password
- Регистрация нового пользователя
- Гостевой вход
- Форма добавления поездки
- Журнал поездок с фильтрами и сортировкой
- Экспорт в CSV
- Роли владельца и ответственных
- Управление списком машин и email-ов ответственных
- PWA-манифест и Service Worker

## Структура проекта

- `index.html` — разметка интерфейса
- `css/app.css` — стили приложения
- `js/app.js` — клиентская логика
- `manifest.json` — PWA-манифест
- `sw.js` — Service Worker
- `FIREBASE_RULES.md` — правила Firebase и настройки

## Запуск

Откройте `index.html` через статический сервер.

Пример:

```bash
python3 -m http.server 4173
```

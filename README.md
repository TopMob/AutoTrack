# AutoTrack

Веб-приложение для ведения журнала поездок

## Возможности

- Вход по Email/Password
- Регистрация нового пользователя
- Гостевой вход
- Форма добавления поездки
- Журнал поездок с фильтрами и сортировкой
- Экспорт в CSV
- Роли владельца и ответственных
- Управление списком машин и email-ов ответственных
- PWA-манифест и Service Worker
